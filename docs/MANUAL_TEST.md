# 手测清单

发版或较大改动后，在 **Windows 10/11** 上按项勾选（源码 `python main.py` 与 `dist/DesktopPet` 各测一轮更佳）。

## 安装与启动

- [ ] `setup_env.bat` 或 `.venv` + `pip install -r requirements.txt` 成功
- [ ] 首次启动出现「选择形象」向导
- [ ] 勾选「下次不再询问」后，再次启动不再弹出向导
- [ ] 托盘 ^ 中能看到图标；启动气泡提示托盘位置

## 无 API

- [ ] API 三项留空可正常进入桌宠
- [ ] 双击聊天提示去「设置」配置
- [ ] 托盘「管理提醒」→「添加提醒」可创建未来提醒
- [ ] 到点托盘通知触发

## 设置与配置

- [ ] 托盘「设置…」修改 API 后聊天可用（无需重启）
- [ ] 修改生存↔普通模式后自动重启，HP 条显示正确
- [ ] 保存 API 后 `.env` 使用 `CHAT_API_*`（旧 `DEEPSEEK_*` 被移除）
- [ ] 托盘「使用说明」「关于」可打开

## 桌宠与形象

- [ ] 宠物显示、随机动作、悬停/拖动动画正常
- [ ] 左键拖动不越界
- [ ] 托盘「居中」有效
- [ ] 「切换形象」后重启且立绘正确
- [ ] 拖文件夹导入形象包

## 生存模式

- [ ] 顶部 HP 条颜色随档位变化
- [ ] 右键「生成食物」→ 拖到宠物喂食，HP 增加
- [ ] 右键「加速一小时」HP 下降
- [ ] 饿死后骷髅；喂满 3 次复活

## 聊天（需 API）

- [ ] 流式回复、停止生成
- [ ] 错误 Key / 错误模型有中文提示
- [ ] 对话设提醒成功；托盘可取消

## 打包 exe

- [ ] `build_release.bat` 成功
- [ ] 解压后双击 `DesktopPet.exe` 可运行
- [ ] 包内含 `首次使用.txt`、`.env.example`

## 自动化

```bash
pip install -r requirements-dev.txt
python -m pytest tests/ -q
```
