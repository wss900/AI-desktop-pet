桌面宠物 · 首次使用（Windows 10/11 64 位）

一、解压与启动
  1. 将整个 DesktopPet 文件夹解压到任意位置（路径尽量不含特殊字符）
  2. 双击 DesktopPet.exe
  3. 首次会弹出「选择形象」界面，按提示操作即可

二、首次向导里可以设置
  · 形象：选一套立绘/GIF
  · 运行模式：
      - 生存模式：顶部 HP、喂食、饿死变骷髅后可复活
      - 普通模式：仅陪伴与聊天，无 HP
  · 聊天 API（OpenAI 兼容，三项都填才能聊天）：
      - API Key
      - API 地址（如 https://api.deepseek.com 或 https://api.openai.com/v1）
      - 模型名称（如 deepseek-chat、gpt-4o-mini）
  · 可先不填 API，仅使用桌宠与提醒；稍后在托盘「设置」里补填

三、之后如何改配置
  · 任务栏右下角托盘 ^ → 右键「桌面宠物」
  · 「设置…」：改模式、API、是否每次启动选形象
  · 「切换形象」：换立绘包（会重启程序）

四、常用操作
  · 双击宠物或托盘「聊天」
  · 托盘「使用说明」：查看玩法与生存规则
  · 托盘「管理提醒」→「添加提醒」：无需 API 也可设闹钟
  · 生存模式：右键宠物可生成食物，拖到宠物身上喂食
  · 拖文件夹到宠物上可导入新形象包

五、若无法启动
  · 安装 VC++ 运行库 x64：
    https://aka.ms/vs/17/release/vc_redist.x64.exe
  · 杀毒软件可能误报 PyInstaller 打包的 exe，可添加信任

六、隐私与安全
  · API Key 只保存在本机 .env，不会上传到作者服务器
  · 聊天记录与记忆保存在本机 data/ 目录
  · 请勿把带真实 Key 的 .env 发给他人或上传到网盘公开链接

更多说明见文件夹内「使用说明.txt」或项目 README。
